##   二、使用方法

#####  1、把maven { url 'https://jitpack.io' }这段代码添加到项目的build.gradle存储库末尾

```java

  allprojects {
        repositories {
              ...
              maven { url 'https://jitpack.io' }
        }
  }

```
#####  2、在需要用的主程序或Module下的build.gradle中引用
```java

  dependencies {
        ...
        implementation 'com.github.barystudio:BSuperView:最新版本'
  }

```
